Escola: representa as instituições cadastradas no sistema, contendo identificação, nome e status, além do vínculo com o pacote contratado.
Pacote: define os planos disponíveis na plataforma, incluindo nome, preço e limite mensal de acessos, podendo estar associado a várias escolas e jogos.
Game: cadastro dos jogos disponíveis na plataforma, contendo nome, descrição e status de ativação, vinculados aos pacotes contratados.
IP_Autorizado: armazena os endereços IP permitidos para acesso de cada escola, reforçando a segurança do sistema.
LOG_Acesso: registra o histórico de acessos realizados, armazenando escola, jogo, IP, data/hora e informação de permissão, permitindo auditoria, relatórios e controle de uso.
